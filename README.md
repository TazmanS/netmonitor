# NetMonitor

NetMonitor is a Linux system monitoring application written in C.

The project collects and displays system and network information from Linux virtual filesystems such as `/proc` and `/sys`.

## Current Features

* CPU temperature
* System uptime
* RAM usage
* Load average
* Network statistics
* Network throughput

## Technologies

* C
* Linux
* Raspberry Pi
* Makefile
* Git

## Build

```bash
make
```

## Run

```bash
./build/netmonitor
```
